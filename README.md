# CarbonBridge

**TurboTax for CBAM** — A DIY CBAM compliance platform for Indian exporters.

**Status:** Prototype / Phase 1 (Core architecture) • Target: MVP in 30 days • Today: Phase 1 setup

---

## TL;DR
CarbonBridge helps small and medium exporters in India gather, validate and export the data required for EU CBAM reporting. This repo holds the frontend (React + Vite) and backend seeds (Supabase + Node) for the project skeleton.

## Tech stack
- Frontend: React + Vite + Tailwind CSS + shadcn/ui
- Backend: Supabase (Auth / Postgres / Storage)
- Hosting: Vercel (frontend) / Supabase (backend)
- PDF export: jsPDF / pdf-lib (planned)
- CI: GitHub Actions
